BIBTEX READ ME FILE

1- Insert bibtex references in the file "libraryIn.bib"
2- After installing bibtool move to the folder "Bibliography"
3- Run the following command in terminal: "bibtool -r bibtoolrsc -i libraryIn.bib -o libraryOut.bib"
3- The bib file "libraryOut.bib" is sorted alphabetically and the entries are cleaned following the format defined in "bibtoolrsc"
